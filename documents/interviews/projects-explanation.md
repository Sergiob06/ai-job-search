# Project Explanations

## BellreguardBasket

### Overview

BellreguardBasket is a full-stack web application developed for a basketball club to manage its public website and administrative tasks.

---

### Main Features

- Team management
- Player management
- Match management
- Statistics
- Image gallery
- Merchandising
- Role-based authentication
- Admin dashboard

---

### Technologies

Backend

- Laravel
- PHP

Frontend

- Blade
- JavaScript
- Tailwind CSS
- Bootstrap

Database

- MySQL

Infrastructure

- Docker
- Linux
- Nginx
- Git
- GitHub

---

### Biggest Challenges

- Designing the database.
- Deploying the application to production.
- Managing user roles and permissions.
- Keeping statistics synchronized with match results.
